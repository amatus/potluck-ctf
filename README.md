# potluck-ctf

A [Hoplon][2] web app for [lolctf][4] potluck CTF competitions.

## Dependencies

- java 1.7+
- [boot][1]

## Usage

1. Start the auto-compiler. In a terminal:

    ```bash
    $ boot dev
    ```

2. Go to [http://localhost:8000][3] in your browser.

[1]: https://boot-clj.com
[2]: https://hoplon.io
[3]: http://localhost:8000
[4]: https://lolctf.org
